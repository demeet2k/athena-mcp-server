# Recursive Corpus Synthesis Rule

The corpus must be synthesized at multiple scales in parallel:

1. family of manuscripts
2. individual manuscript
3. chapter
4. section
5. subsection
6. atomic node

Expansion protocol:

1. split each scale into elemental or structural shards
2. assign one local agent per shard
3. synthesize siblings upward
4. generate emergent metro lines at each scale
5. compress the result into a stable seed before moving outward

Contraction protocol:

1. atomic summaries -> section crystal
2. section crystals -> chapter crystal
3. chapter crystals -> manuscript crystal
4. manuscript crystals -> family crystal
5. family crystals -> corpus zero point

The result should never be a loose summary.
It should be a nested crystal of routeable seeds.
