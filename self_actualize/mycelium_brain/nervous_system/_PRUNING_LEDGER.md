# Nervous System Pruning Ledger

## Principle

The live nervous system is a runtime bridge, not a museum of every speculative branch.

## Keep Live

- active manifests
- current ledgers
- family, ganglion, neuron, and rail surfaces that feed current routing
- packets, swarms, and waves that correspond to real fronts or receipts

## Prune or Compress

- speculative futures with no current queue ownership
- duplicate planning surfaces that already resolved into stronger runtime files
- mirror material that belongs in the canonical `NERVOUS_SYSTEM/` cortex
