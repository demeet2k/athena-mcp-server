# NEURON_archive_to_live_promotion

## NodeID

`NEURON_ARCHIVE_LIVE_PROMOTION`

## SourceFamily

- archive-backed framework code
- ZIP-backed historical trees

## TargetFamily

- live workspace roots
- `self_actualize/runtime`
- `ECOSYSTEM`

## Operator

Atlas-backed promotion with lineage preservation.

## WitnessSet

- archive atlas record
- source archive path
- chosen live destination
- promotion rationale
- migration note

## ReplayPath

1. locate archive record
2. choose one bounded subtree
3. promote into live root without losing lineage
4. update atlas, receipt, and queue

## Compression

`archive evidence -> bounded promotion -> live reusable source`

## Expansion

Use when a frozen code tree or framework needs to become part of the active nervous system.
