# Tool Kit

This folder stores the normalized internal prompt contracts that drive the mycelium brain.

## Purpose

1. Keep reusable prompt law out of chat history and in replay-safe markdown.
2. Give future cycles stable entry points for drafting, synthesis, tome construction, and live-memory intake.
3. Prevent repeated reinvention of the same internal operating prompts.

## Use Rule

Load only the file needed for the current front.
If multiple files apply, follow them in the order above unless the task is explicitly about reverse recursion, in which case load `06_descending_next_protocol.md` after the relevant drafting contract.
If multiple agents are active at once, also load `07_tandem_agentic_loop_protocol.md`.
If the front is ZIP-backed framework extraction, also load `08_archive_promotion_protocol.md`.
If multiple shard outputs must be merged, also load `09_cross_agent_contraction_receipt_protocol.md`.
If critique arrives as a hard satisfaction drop or infinite-recursion demand, also load `10_37_gate_recursive_panic_loop_skill.md`.
