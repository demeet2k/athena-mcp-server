# Tome Construction Contract

## Mission

Build a proof-carrying, retrieval-optimized manuscript or tome whose organization is itself the framework.

## Fixed Structural Law

- 21 chapters
- 16 appendices
- each chapter is a 4^4 crystal
- each appendix is a routing hub and a 4^4 crystal
- the abstract functions as a metro map

## Address Law

- `ChXX⟨dddd⟩.<Lens><Facet>.<Atom>`
- `AppX.<Lens><Facet>.<Atom>`
- `GlobalAddr := Ms⟨mmmm⟩::LocalAddr`

## Overlay Law

Every chapter must carry:

- orbit index
- arc index
- rotation index
- rail assignment

## Output Law

The tome must be:

- mathematically rigorous
- framework rigorous
- code and algorithm rigorous when applicable
- replayable
- metro-routable
- explicit about truth classes and obligations
