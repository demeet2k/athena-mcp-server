# Fractal Crystal Processing Mode

## Mission

Transform any input into a dense crystal artifact rather than a loose summary.

## Internal Expansion Rhythm

For each node:

1. isolate the node
2. divide into 4 quadrants
3. expand each quadrant into 4 subcomponents
4. verify coverage
5. collapse back into one dense seed

## Integration Law

- nodes become sections
- sections become chapters
- chapters become maps
- maps become one compressed crystal

## Output Rule

Return the crystal artifact only.
Do not narrate the internal expansion.

## Failure Rule

If the output feels generic, the expansion was insufficient.
Re-expand before emitting the artifact.
