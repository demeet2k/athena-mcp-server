# deeper-network-basis-router

Use when a task touches the deeper integrated network and must be grounded in the correct basis documents before analysis starts.

## Workflow

1. Read `../../00_CONTROL/04_ALGORITHMIC_PIPELINE.md`.
2. Check `../../10_LEDGERS/01_CANONICAL_SOURCES.md`.
3. Read `../00_SKILL_ROUTER.md`.
4. If needed, inspect `../../05_MATRIX_16X16`, `../../07_METRO_STACK`, and `../../08_APPENDIX_CRYSTAL`.
5. If the request reaches Level `5-7`, `H6`, `Seed-6D`, or `NEXT^[4^6]`, hand off to `lane-stack-orchestrator`.

## Output contract

- Name the basis documents.
- Name the dimension stage.
- Name the active element or symmetry.
- Name the metro level.
- Name the appendix floor.
