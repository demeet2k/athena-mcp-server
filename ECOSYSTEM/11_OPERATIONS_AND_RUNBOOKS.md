# OPERATIONS AND RUNBOOKS

## 1. Corpus Ingestion

Steps:
1. Identify source document.
2. Extract definitions and axioms.
3. Map to addressing scheme.
4. Assign truth class.

## 2. Synthesis Run

Steps:
1. Expand tasks via agent crystal.
2. Contract outputs into final artifact.
3. Record witness and replay steps.

## 3. Release

Steps:
1. Validate against QA checklist.
2. Update registry and version.
3. Publish with migration notes.

## 4. Incident Response

- If contradiction detected, issue CONFLICT edge.
- If unverifiable, mark FAIL and quarantine.
- If ambiguous, mark AMBIG and plan evidence.

## 5. Status
This runbook governs daily operations for the ecosystem.
