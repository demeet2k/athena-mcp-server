# FRACTAL CRYSTAL ENGINE PROTOCOL

## Core Law

Do not process a source as a loose summary when a crystal artifact is required.

## Processing Discipline

For each structural unit:

1. isolate the unit,
2. divide it into four conceptual quadrants,
3. expand each quadrant into four subcomponents,
4. test for gaps,
5. collapse the expanded matrix back into a high-density seed,
6. move to the next unit and then synthesize upward.

## Scaling Law

Apply the same expansion and contraction at:

- sub-section scale,
- section scale,
- chapter scale,
- appendix scale,
- whole-manuscript scale.

## Output Standard

The result should function as a seed that can be re-expanded later without losing the
main structure.
