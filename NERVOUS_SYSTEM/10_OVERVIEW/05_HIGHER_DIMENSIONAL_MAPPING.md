# HIGHER-DIMENSIONAL MAPPING

## Purpose

The nervous system cannot be represented only as a 4-element crystal. It also moves through a transport manifold, orbit phases, control rails, swarm depths, truth classes, packet types, and cooperating surfaces.

## Coordinate Bundle

Define the live nervous-system coordinate:

`NSCoord = (Addr4, Face6, Arc, Rail, Depth, Packet, Truth, Surface)`

where:

- `Addr4` is the 4-element lineage address
- `Face6` is the active transport face in the 6-face manifold
- `Arc` is the Circle orbit phase
- `Rail` is the Triangle control rail
- `Depth` is swarm or recursion depth
- `Packet` is the active packet class
- `Truth` is corridor truth
- `Surface` is one of `{Cortex, RuntimeHub, GovernanceMirror}`

This bundle is the minimum lawful location of a live nervous-system event.

## Crystal Layer

The crystal layer remains:

- Earth
- Water
- Fire
- Air

It governs role, local work type, contraction precedence, and synthesis style.

## Transport Manifold

The transport manifold adds:

- Earth
- Air
- Water
- Fire
- Void
- Aether

The extra axis is the opposition:

- `Void`
- `Aether`

This gives the nervous system a reset and integration axis that the flat crystal alone cannot express.

### Face Meaning

- `Earth`: storage, fixation, grounding
- `Air`: map, abstraction, possibility
- `Water`: continuity, linkage, flow
- `Fire`: execution, pressure, transformation
- `Void`: reset, deletion of stale policy shells, non-loaded capsule state
- `Aether`: integrated transport, high-coherence loaded state

### Routing Consequence

No opposite transition is treated as direct:

- Earth <-> Air requires a midpoint
- Water <-> Fire requires a midpoint
- Void <-> Aether requires a midpoint

Every major nervous-system transition should therefore be routed through hybrid faces or named bridges rather than asserted as instantaneous.

## Circle and Triangle Overlay

The higher-dimensional nervous system inherits:

- `Arc`
- `Rot`
- `Rail`

Operationally:

- Circle determines macro-phase and revisit order
- Triangle determines long-rail affinity and control semantics

These are not cosmetic overlays. They determine which fronts should exchange frequently and which should exchange only through transfer hubs.

## Swarm Depth Axis

The scale axis is:

- `D0`: kernel
- `D1`: pillars
- `D2`: pantheon
- `D3`: workers
- `D4`: microcells

Typical sizes:

- `1`
- `4`
- `16`
- `64`
- `256`

The deeper swarm begins when 64 workers are no longer enough and explicit microcell specialization is required.

## Packet Axis

The packet layer determines how a coordinate behaves:

- task packet
- worker packet
- pod packet
- ganglion packet
- wave packet
- synthesis packet
- receipt packet
- queue packet
- neuron packet

The same region may therefore be revisited under different packet classes without ambiguity.

## Truth Axis

Every coordinate is typed by:

- `OK`
- `NEAR`
- `AMBIG`
- `FAIL`

This prevents the system from confusing mapped, proven, partially synthesized, and blocked regions.

## Surface Axis

The surface axis names where the event is instantiated:

- `Cortex`: canonical root `NERVOUS_SYSTEM/`
- `RuntimeHub`: `self_actualize/mycelium_brain/`
- `GovernanceMirror`: `ECOSYSTEM/NERVOUS_SYSTEM/`

The same front may appear on more than one surface, but each surface should have a different responsibility.

## Example Reading

`(F-A-W, Aether, Arc3, Me, D3, worker_packet, NEAR, RuntimeHub)`

means:

- lineage address `F-A-W`
- integrated loaded state
- arc 3
- mercury rail
- worker depth
- worker packet
- structurally useful but not yet fully witnessed
- instantiated in the live runtime hub

## Zero Point

The zero point is not one folder. It is the smallest stable bundle that can regenerate the rest:

`NS0 = (kernel, Aether, current arc, current rail, D0, task_packet, NEAR or OK, RuntimeHub)`

If this bundle survives, the nervous system can reopen without pretending the entire old state must be preserved.
