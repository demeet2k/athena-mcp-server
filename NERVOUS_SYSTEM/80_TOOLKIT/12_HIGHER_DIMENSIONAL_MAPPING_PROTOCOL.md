# HIGHER DIMENSIONAL MAPPING PROTOCOL

## Mission

Map every promoted artifact into a neuron coordinate that preserves not just file location but crystal lineage, rail ownership, orbit position, hub dependencies, truth class, and manuscript family.

## Minimum Coordinate

Use:

NeuronAddr = <F, M, Ch, L, Fc, At, G1, G2, G3, G4, Arc, Lane, Truth, Hub>

Where:
- F = family
- M = manuscript
- Ch = chapter or source station
- L = lens
- Fc = facet
- At = atom
- G1..G4 = 4-level lineage in E/W/F/A space
- Arc = orbit arc
- Lane = Su / Me / Sa
- Truth = OK / NEAR / AMBIG / FAIL
- Hub = dominant appendix hub

## Projection Rules

1. Local manuscript view projects to <Ch, L, Fc, At>.
2. Swarm execution view projects to <G1, G2, G3, G4>.
3. Routing view projects to <Arc, Lane, Hub, Truth>.
4. Corpus view projects to <F, M>.

## Law

No promoted artifact should remain only a file path if it is intended to participate in the nervous system.
