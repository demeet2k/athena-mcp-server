# Record Class And Surface Class Map

## Surface Classes

| Surface Class | Meaning |
| --- | --- |
| atlas | record substrate and cross-index surfaces |
| registry | typed row stores and machine registries |
| dashboard | state summaries and front-door status maps |
| overview | human-readable doctrinal or structural overviews |
| capsule | domain contractions and source body families |
| graph | edge, neuron, synapse, and bridge tissue |
| runtime lane | verification, execution, or routed runtime lanes |
| queue | ordered next-work and regeneration surfaces |
| packet | bounded traversal or transform payloads |
| mirror | runtime or governance reflection surfaces |
| schema | formal interface surfaces preserved explicitly |

## Artifact Classes

| Artifact Class | Meaning |
| --- | --- |
| manuscript | text-heavy source, chapter, appendix, or book matter |
| code | executable scripts, modules, or configurations |
| data | json, csv, yaml, or structured payloads |
| media | images, pdfs, binaries, or non-text witness bodies |
| generated | machine-derived runtime or dashboard artifacts |
| ledger | receipts, progress ledgers, and proof-carrying traces |
| manifest | active fronts, registries, queue summaries, and maps |
| schema | formal interface or law declarations |
| receipt | completion traces and derivation writebacks |
| queue | ordered next-work surfaces |
| skill | local skill definitions and routing doctrine |

## Truth Roles

| Truth Role | Meaning |
| --- | --- |
| source | primary local source matter |
| generated | derived from runtime or control-plane generation |
| protocol | normative law, contract, or procedure |
| ledger | proof trail or progress witness |
| receipt | writeback and closure proof |
| mirror | runtime or governance reflection |
| archive-backed | indexed through archive witness rather than live file path |
| reserve | kept visible without forcing false integration claims |

## Replay Classes

| Replay Class | Meaning |
| --- | --- |
| replay-safe | may be revisited and regenerated directly |
| replay-partial | requires extra witness or writeback context |
| generated-refreshable | safe to rebuild from source surfaces |
| witness-only | presence can be cited but not fully replayed as text |
