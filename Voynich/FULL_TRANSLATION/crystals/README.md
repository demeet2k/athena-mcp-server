# Voynich Crystal Suite

## Purpose

These files are the section-level and corpus-level crystal documents for the final translation program.

Each crystal file contains:

1. synthesis
2. metro map
3. emergent metro map
4. current zero point
5. growth queue

## Build Rule

After each authoritative folio is finished:

- update the relevant crystal file
- update the unified corpus
- update the working metro map
- update the build status
