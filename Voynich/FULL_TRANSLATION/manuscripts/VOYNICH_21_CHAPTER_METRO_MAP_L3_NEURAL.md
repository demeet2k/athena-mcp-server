# Level 3 Neural Metro Map

## Purpose

This is the high-resolution neighborhood map. It treats the chapter system like a firing network and groups chapters by recurrent activation clusters instead of by table-of-contents proximity alone.

## Clusters

| Cluster | Stations | Meaning |
| --- | --- | --- |
| Seed Cluster | `1, 2, 3, 4` | axioms, decomposition, algebra, convergence |
| Activation Cluster | `5, 9, 10, 11, 15, 17, 19` | all chapters where payload is heated, timed, bounded, or certified |
| Circulation Cluster | `6, 9, 12, 13, 15, 16, 18, 19` | all chapters where loops and settling dominate |
| Address Cluster | `7, 10, 12, 14, 15, 17, 18, 19` | all chapters where indexing and lawfulness fire |
| Containment Cluster | `8, 11, 13, 14, 16, 17, 18, 19` | all chapters where topology and proof hold the machine |
| Lift Cluster | `4, 19, 20, 21` | all chapters where compression and next-dimensional motion dominate |
| Shadow Cluster | `3, 11, 16, 21` | chapters exposing failure, missingness, imbalance, or frontier |
| Replay Cluster | `3, 8, 19, 20, 21` | chapters that determine whether the structure can be regenerated |

## Neural Hubs

- `Ch 15` = Activation Cluster + Circulation Cluster + Address Cluster
- `Ch 18` = Circulation Cluster + Address Cluster + Containment Cluster
- `Ch 19` = Activation Cluster + Circulation Cluster + Address Cluster + Containment Cluster + Lift Cluster + Replay Cluster

## Zero-Point Interpretation

At neural resolution the center is not only `Ch 19`, but the `Ch 19 / Ch 20 / Ch 21` triad where omega, application, and frontier continually refire each other.
