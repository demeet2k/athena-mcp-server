## 4 x 4 matrix

- `runtime.route.file.seed`: Use `route` on the single source artifact until it can define the object and create the first admissible representation.
- `runtime.route.file.link`: Use `route` on the single source artifact until it can connect the object to parents, peers, and downstream surfaces.
- `runtime.route.file.prove`: Use `route` on the single source artifact until it can add tests, receipts, witnesses, or quantitative closure.
