# Runtime Replay

## Role

## Targets

- self_actualize
- NERUAL NETWORK
- Trading Bot
- PoleStarGEMM

## 4 x 4 matrix

## File scale

- `runtime.replay.file.seed`: Use `replay` on the single source artifact until it can define the object and create the first admissible representation.
- `runtime.replay.file.link`: Use `replay` on the single source artifact until it can connect the object to parents, peers, and downstream surfaces.
- `runtime.replay.file.prove`: Use `replay` on the single source artifact until it can add tests, receipts, witnesses, or quantitative closure.
- `runtime.replay.file.publish`: Use `replay` on the single source artifact until it can promote the object into canonical operating surfaces.
