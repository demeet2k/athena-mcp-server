# Corpus Replay

## Role

Live manuscripts, markdown mirrors, notes, and visible source files.

## Targets

- DEEPER CRYSTALIZATION
- MATH
- Voynich
- FRESH

## 4 x 4 matrix

## File scale

- `corpus.replay.file.seed`: Use `replay` on the single source artifact until it can define the object and create the first admissible representation.
- `corpus.replay.file.link`: Use `replay` on the single source artifact until it can connect the object to parents, peers, and downstream surfaces.
- `corpus.replay.file.prove`: Use `replay` on the single source artifact until it can add tests, receipts, witnesses, or quantitative closure.
- `corpus.replay.file.publish`: Use `replay` on the single source artifact until it can promote the object into canonical operating surfaces.
