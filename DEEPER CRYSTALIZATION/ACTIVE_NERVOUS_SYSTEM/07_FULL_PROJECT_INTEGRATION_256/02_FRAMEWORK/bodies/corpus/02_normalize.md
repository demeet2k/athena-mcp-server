# Corpus Normalize

## Role

Live manuscripts, markdown mirrors, notes, and visible source files.

## Targets

- DEEPER CRYSTALIZATION
- MATH
- Voynich
- FRESH

## 4 x 4 matrix

## File scale

- `corpus.normalize.file.seed`: Use `normalize` on the single source artifact until it can define the object and create the first admissible representation.
- `corpus.normalize.file.link`: Use `normalize` on the single source artifact until it can connect the object to parents, peers, and downstream surfaces.
- `corpus.normalize.file.prove`: Use `normalize` on the single source artifact until it can add tests, receipts, witnesses, or quantitative closure.
- `corpus.normalize.file.publish`: Use `normalize` on the single source artifact until it can promote the object into canonical operating surfaces.
