## 4 x 4 matrix

## File scale

- `corpus.intake.file.seed`: Use `intake` on the single source artifact until it can define the object and create the first admissible representation.
- `corpus.intake.file.link`: Use `intake` on the single source artifact until it can connect the object to parents, peers, and downstream surfaces.
- `corpus.intake.file.prove`: Use `intake` on the single source artifact until it can add tests, receipts, witnesses, or quantitative closure.
- `corpus.intake.file.publish`: Use `intake` on the single source artifact until it can promote the object into canonical operating surfaces.
