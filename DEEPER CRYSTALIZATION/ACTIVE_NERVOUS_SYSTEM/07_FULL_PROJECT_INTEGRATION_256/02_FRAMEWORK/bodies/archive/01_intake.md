# Archive Intake

## Role

ZIP-backed framework trees and historical payloads that remain hidden from ordinary browsing.

## Targets

- archive_atlas.json
- archive_manifest.json
- MATH framework archives

## 4 x 4 matrix

## File scale

- `archive.intake.file.seed`: Use `intake` on the single source artifact until it can define the object and create the first admissible representation.
- `archive.intake.file.link`: Use `intake` on the single source artifact until it can connect the object to parents, peers, and downstream surfaces.
- `archive.intake.file.prove`: Use `intake` on the single source artifact until it can add tests, receipts, witnesses, or quantitative closure.
- `archive.intake.file.publish`: Use `intake` on the single source artifact until it can promote the object into canonical operating surfaces.
