# ATLAS LAYER

This subfolder contains machine-readable and support artifacts for the global distillation.

## Files

- `corpus_stats.json` - generated manifest of corpus size, body counts, extension counts, representative files, and sampled term presence

## Intended Use

Use this layer when you need:

1. corpus evidence before synthesis,
2. machine-readable body counts,
3. a quick witness check for how the global folder was grounded.

The markdown files in the parent folder are the human-readable distillation. The JSON here is the witness substrate.
